# BCA Final Year Project

## Project Title
Student Record Management System using Python

## Abstract
This project is developed as part of the BCA final year curriculum.
The system helps to manage student records efficiently using Python
and SQLite database without using HTML, IoT, or PHP.

## Technologies Used
- Python
- SQLite

## Modules
- Add Student
- View Student
- Update Student
- Delete Student
